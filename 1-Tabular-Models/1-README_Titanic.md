# 🚢 Titanic Survival Prediction: Neural Engine vs. Traditional ML

**Technical Status:** Fully Deployed & Operational  
**Architecture:** Keras Deep Learning & Scikit-Learn Ensemble Comparison

This project is a core module of our advanced AI ecosystem, focusing on binary classification under data scarcity. It evaluates the performance gap between traditional machine learning algorithms and a custom-built Neural Engine.

### ⚙️ Pipeline Architecture (10-Step Production Pipeline)
This project strictly follows our **10-Step Production Pipeline**:
1. **Objective Definition:** Binary classification of passenger survival.
2. **Data Ingestion & EDA:** Visualizing survival rates across classes and ages.
3. **Feature Engineering:** Creating the 'Age*Class' artifact for higher correlation.
4. **Data Cleaning:** Intelligent imputation of missing 'Age' and 'Embarked' data.
5. **Model Training:** Comparative analysis of Gradient Boosting vs. Multi-Layer Perceptrons (MLP).
6. **MLOps Standard:** Model serialization (.h5 / .pkl) and production-ready deployment.

### 🌐 Live Predictive Interface
The survival engine is fully deployed. Users can input passenger metrics and receive real-time survival probabilities.
