🚀 Spaceship Titanic: Multi-Dimensional Neural Engine
Technical Status: Fully Deployed & Operational

Architecture: Keras Deep Learning (MLP) with Automated Missing Value Imputation

This project is an advanced iteration of the classic Titanic problem, set in a high-dimensional space-travel dataset. It focuses on predicting which passengers were transported to an alternate dimension using a custom-built Neural Engine. The core challenge involves handling complex missing data patterns across multi-categorical features.

🧠 The Engineering Philosophy: The 10-Step Pipeline
Following our signature 10-Step Production Pipeline, this project demonstrates how to handle sparse datasets with deep learning:

Objective Definition: Predicting passenger transport status (Binary Classification) in a multi-feature space environment.

Data Ingestion & EDA: Investigating the high correlation between CryoSleep status and transportation rates.

Feature Selection: Isolating predictive features like PassengerId (Group-level) and Spending habits.

Categorical to Numerical: Encoding sci-fi features like HomePlanet and Destination.

Data Cleaning: Implementing advanced imputation strategies for missing values in VIP, Age, and Luxury Amenities.

Advanced Feature Engineering: Synthesizing 'Total Spending' and extracting 'Deck/Room' information from Cabin data.

One-Hot Encoding & Scaling: Optimizing input tensors for the Neural Engine using StandardScaler.

Train/Test Splitting: Securing a robust validation set to monitor over-fitting.

Model Training (Neural Engine): A deep Multi-Layer Perceptron (MLP) architecture built with Keras, utilizing ReLU activation and Adam optimizer.

Evaluation & Serialization: Achieving a 0.805 accuracy score and serializing the model for cloud deployment.

🌐 Live Predictive Interface
The interstellar transport prediction engine is fully deployed. Users can input passenger credentials—including cabin location and luxury spending—to calculate transportation probabilities.
