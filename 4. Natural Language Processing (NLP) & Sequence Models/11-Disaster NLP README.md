## 🏗️ Project Vision: The "AI Factory"
This project is not just a simple classification model; it is a complete **MLOps pipeline**. The architecture is designed as a scalable factory that bridges the gap between raw data and real-world deployment. 

> "I engineered this system to be an autonomous infrastructure. By focusing on vectorized inference and a rock-solid data pipeline, the engine is ready to scale from synthetic testing to high-power GPU production in seconds."

## 🧠 Technical Architecture
The core intelligence of this engine relies on a **Bidirectional LSTM (Long Short-Term Memory)** network, allowing the model to understand the context of a sentence from both directions—ensuring it doesn't get confused by metaphorical language.



### Key Features:
* **Vectorized Inference:** Optimized data flow with zero-loop processing for high-speed predictions.
* **Modern Keras Engine:** Utilizing the `.keras` native format to ensure seamless transition between training and production environments.
* **Linguistic Context Awareness:** Capable of distinguishing between real emergencies (e.g., "The building is on fire!") and metaphorical expressions (e.g., "That movie was fire!").
* **MLOps Ready:** A structured pipeline designed for continuous integration and deployment.
