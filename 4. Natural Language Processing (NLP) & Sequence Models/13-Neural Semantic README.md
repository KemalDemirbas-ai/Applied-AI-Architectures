## 🚀 Overview
The **Neural Semantic Engine** is an advanced NLP module built to identify semantic similarity and contextual overlap between text pairs. Utilizing a **Multi-Input Siamese Bidirectional LSTM** architecture, the system maps questions into a high-dimensional vector space to detect intent duplication with industrial precision.

This project was developed as a core component of a larger **Autonomous AI Ecosystem**, focusing on contextual logic and data redundancy elimination.

## 🛠️ Technical Architecture & MLOps Pipeline
This project follows a strict **10-Step MLOps Compliance** framework:

1.  **Objective Definition:** Targeted binary classification for contextual intent.
2.  **Exploratory Data Analysis (EDA):** Deep dive into class imbalance and sequence density.
3.  **Data Cleaning:** Regex-based optimization for noise reduction and null handling.
4.  **Feature Engineering:** Analysis of word length distributions and semantic windowing.
5.  **Data Encoding:** Unified vocabulary space mapping via integer tensors.
6.  **Tensor Transformation:** High-speed neural-ready data pipelines.
7.  **Data Splitting:** 15% strict isolation policy for unbiased validation metrics.
8.  **Model Design:** Shared-weight Siamese layers using Manhattan distance mapping.
9.  **Training:** Adaptive learning with weight sharing and loss convergence tracking.
10. **Cloud Deployment:** Scalable inference via Streamlit on Hugging Face Spaces.
