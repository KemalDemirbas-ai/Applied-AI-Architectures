## 🧠 Project Overview
Cassava is a critical food security crop. This project leverages **Transfer Learning** to provide instant disease diagnosis, helping optimize agricultural yields.

### Key Technical Features:
* **Architecture:** Fine-tuned **EfficientNetB0** for high-precision image classification.
* **Dynamic Pipeline:** Implemented custom class weighting and dynamic learning rates to handle dataset imbalances.
* **Preprocessing:** Custom vision engine that handles RGB conversion and alpha-channel filtering for "in-the-wild" images.
* **Interface:** A mobile-responsive web app built with **Streamlit**.
